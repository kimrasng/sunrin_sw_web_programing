HTML 구조
    입력 필드 (<input id="display">): 결과와 입력값 표시.
    숫자 버튼 (.number): 0~9 숫자 입력.
    연산자 버튼 (.operator): +, -, *, /, . 입력.
    등호 버튼 (#equals): 수식 계산.
    지우기 버튼 (#clear): 마지막 문자 삭제.
    전체 지우기 버튼 (#clearAll): 전체 입력 삭제.
JavaScript 코드
    숫자 버튼 클릭: 클릭된 숫자를 디스플레이에 추가.
    연산자 버튼 클릭: 클릭된 연산자를 디스플레이에 추가.
    등호 버튼 클릭: 디스플레이의 수식을 평가하고 결과 표시.
    지우기 버튼 클릭 (C): 마지막 문자 삭제.
    전체 지우기 버튼 클릭 (AC): 전체 입력 삭제.
    수식 평가 (evaluateExpression 함수): 수식을 안전하게 평가하여 결과 반환.
